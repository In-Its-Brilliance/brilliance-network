Интерактивный чат через сетевой слой.

Сервер принимает подключения клиентов. Клиенты отправляют текстовые сообщения через консоль, сервер рассылает их всем подключённым клиентам.

```shell
# Терминал 1 - сервер
cargo run -p network-chat -- -t=server --ip=192.168.0.185:25565

# Терминал 2 - клиент
cargo run -p network-chat -- -t=client --ip=192.168.0.185:25565

# Сборка под Windows
cargo b -p network-chat --release --target x86_64-pc-windows-gnu
```
